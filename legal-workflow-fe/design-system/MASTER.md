# Legal Workflow - Design System MASTER
> Generated by UI UX Pro Max Skill | Target: Legal Workflow (React + Tailwind CSS v4)

---

## 1. Design Philosophy

**Pattern:** Trust & Authority + Minimalism
**Style:** Trust & Authority (WCAG AAA, Tailwind 10/10)
**Mood:** Professional, Authoritative, Trustworthy, Formal

**Core Principles:**
- Credibility paramount - every pixel communicates trust
- Data clarity > decoration
- Accessibility first (WCAG AAA, keyboard nav, reduced-motion)
- Minimal cognitive load - clean hierarchy, functional colors

**Anti-Patterns (AVOID):**
- Outdated design patterns
- Hidden credentials or status
- AI purple/pink gradients
- Excessive decoration, complex shadows, 3D effects
- Emoji as structural icons (SVG only)

## 4. Icons

### Library: Heroicons (current) / Phosphor (recommended upgrade)

**Current:** Inline SVG Heroicons (outline, 24x24, strokeWidth 1.5)
**Recommended:** `@phosphor-icons/react` for richer legal-specific icon set

### Icon Rules
- No emoji as structural icons (SVG only)
- Consistent 20px sidebar, 16-20px inline
- `shrink-0` to prevent flex squeeze
- `currentColor` for theme inheritance

## 5. Spacing & Layout

### 8dp Rhythm
- Base unit: 8px (0.5rem)
- Spacing scale: 4, 8, 12, 16, 24, 32, 48, 64
- Card padding: 24px (p-6)

## 7. Interactions

| Element | Duration | Property |
|---------|----------|----------|
| Buttons | 150ms | `transition-colors` |
| Cards | 200ms | `transition-all` |
| Links | 150ms | `transition-colors` |
| Nav items | 150ms | `transition-all` |

### Focus States
```css
*:focus-visible { outline: 2px solid #1E3A8A; outline-offset: 2px; border-radius: 4px; }
```

### Reduced Motion
```css
@media (prefers-reduced-motion: reduce) { *, *::before, *::after { animation-duration: 0.01ms !important; transition-duration: 0.01ms !important; } }
```
